# Manual de Usuario

## Que es Quest4Quill

Quest4Quill es una aplicacion para organizar mundos narrativos. Te permite crear varios mundos y, dentro de cada uno, gestionar personajes, notas, historias, relaciones, lugares, organizaciones y objetos.

La informacion se guarda de forma local en el equipo.

## Pantalla completa de la aplicacion

Pulsa `F11` para activar o desactivar la pantalla completa borderless de la ventana de la aplicacion.

Este modo:
- ocupa toda la pantalla disponible
- oculta la decoracion de la ventana
- es util para trabajar con mas espacio visual

## Pantalla inicial

Al abrir la aplicacion llegas a la biblioteca de mundos.

Desde aqui puedes:
- abrir un mundo existente
- crear un mundo nuevo
- ver de un vistazo los proyectos que ya tienes

### Crear un mundo

1. Pulsa `Crear Nuevo Mundo`.
2. Rellena el nombre del mundo.
3. Confirma la creacion.

El mundo aparecera en la biblioteca y quedara disponible para abrirlo despues.

## Entrar en un mundo

Pulsa sobre cualquier tarjeta de mundo para abrirlo.

Una vez dentro veras:
- el nombre del mundo arriba
- la barra lateral con las secciones
- el contenido de la seccion seleccionada en el panel principal

## Barra lateral

La barra lateral organiza el contenido por categorias:

- `Historias`
- `Notas`
- `Relaciones`
- `Personajes`
- `Organizaciones`
- `Regiones`
- `Ubicaciones`
- `Objetos`

### Plegar y desplegar la barra lateral

Puedes usar el boton de la parte superior de la barra lateral para ocultarla y ganar espacio.

En pantallas pequenas la barra puede compactarse automaticamente.

### Volver a mundos

El boton `Volver a mundos` te lleva a la biblioteca principal.

## Historias

La seccion de historias sirve para organizar el contenido narrativo por tramas, capitulos o bloques de historia.

Usala para:
- crear historias
- revisar el material narrativo
- moverte entre escenas o capitulos segun la estructura que uses

## Notas

La seccion de notas esta pensada para ideas sueltas, recordatorios, lore, referencias o cualquier texto que quieras tener a mano.

## Personajes

La seccion de personajes te permite crear y editar fichas completas.

Puedes guardar:
- nombre
- apellidos
- titulo
- alias
- genero
- edad
- estado
- tipo de personaje
- notas
- descripcion
- motivaciones
- personalidad
- residencia
- objetos asociados
- relaciones

### Vista de lista

Los personajes aparecen agrupados por tipo:
- principales
- mayores
- menores

### Vista de detalle

Al abrir un personaje veras su ficha completa.

Desde esa ficha puedes:
- editar sus datos
- eliminarlo
- volver a la lista
- revisar sus relaciones

### Edicion de relaciones

En la ficha de un personaje puedes crear, editar o borrar relaciones con otros personajes.

Tipos disponibles:
- progenitor
- descendiente
- hermano/a
- familia
- amigo
- amante
- rival

## Organizaciones

La seccion de organizaciones permite registrar grupos, facciones, gremios o instituciones.

Normalmente se usa para:
- nombre de la organizacion
- descripcion
- lideres
- miembros
- informacion complementaria

## Regiones y ubicaciones

Estas secciones sirven para estructurar el mapa del mundo.

### Regiones

Usalas para zonas amplias, paises, continentes o territorios grandes.

### Ubicaciones

Usalas para sitios mas concretos dentro de una region:
- ciudades
- pueblos
- edificios
- zonas especificas

## Objetos

La seccion de objetos sirve para registrar artefactos, herramientas, reliquias o cualquier elemento importante de la historia.

Puedes consultar:
- nombre
- tipo
- descripcion
- localizacion
- portador

## Relaciones

La seccion de relaciones muestra un mapa interactivo con personajes y objetos conectados entre si.

### Que puedes hacer en el mapa

- acercar y alejar con `+` y `-`
- usar la rueda del raton para zoom
- mover el fondo arrastrando en un espacio vacio
- arrastrar nodos para reorganizar el grafo
- abrir la vista expandida para ver el mapa con mas espacio
- volver a la vista original con `Vista original`

### Boton de pantalla completa

El boton con icono de expandir activa una vista ampliada del mapa.

En ese modo:
- se oculta la parte superior de la vista
- desaparece el panel de detalle lateral
- el mapa ocupa casi toda la pantalla disponible

Pulsa `Esc` o el mismo boton otra vez para salir.

### Arrastrar nodos

Los nodos se pueden mover un poco para reorganizar la vista.

El comportamiento es elasticos o con efecto muelle:
- el nodo sigue el puntero con suavidad
- el resto del grafo se reajusta
- al soltarlo, la red termina de asentarse

## Guardado de datos

La aplicacion guarda la informacion de forma local.

Eso significa:
- no necesitas guardar manualmente cada cambio
- los datos se conservan al cerrar la app
- si borras los datos del navegador embebido, tambien puedes perder los mundos

## Consejos de uso

- Empieza creando primero un mundo y luego completa personajes, notas y relaciones.
- Usa el mapa de relaciones cuando tengas suficientes nodos para entender mejor la red entre elementos.
- Si una vista se siente pequena, prueba el modo expandido del mapa.
- Si trabajas en movil o ventanas pequenas, la barra lateral puede ocultarse para dejar mas espacio.

## Problemas comunes

### No veo mi mundo

Comprueba que estas en la biblioteca inicial y que el mundo se creo correctamente.

### No aparecen nodos en Relaciones

Necesitas tener personajes u objetos creados para que el mapa muestre contenido.

### La vista parece muy cargada

Usa:
- zoom
- arrastre del fondo
- vista expandida
- reordenacion manual de nodos

## Resumen rapido

1. Crea o abre un mundo.
2. Rellena personajes, notas, lugares y objetos.
3. Usa Relaciones para conectar y visualizar el universo.
4. Navega entre secciones desde la barra lateral.
